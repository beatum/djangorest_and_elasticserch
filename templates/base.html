  • Don't foget create many article with command: ./manage.py create_article 1000
  • API list: /api/v1/article/ - Here you can test the sampling method on the upper boundary of the field DateTimeTZRange (active_range)
  • API search: /api/v1/search/ - Full-text search

    How to search? For example:

    • /api/v1/search/?title__contains=lorem - use regular django query (startswith, exact, iexact, contains, lt, gt, lte, gte, etc...)
    • /api/v1/search/?title__contains=lorem&text__contains=ipsum - —Åhain of parameters
    • /api/v1/search/?title__contains=lorem&boost=ipsum,1.1 - boosting by feild relevance (by default 'title' is more relevance then 'text')
    • /api/v1/search/?format=json&limit=100&offset=100&text__contains=ipsum&title__contains=lorem - finally you can use all together with limiting and offsetting, with output in json format
